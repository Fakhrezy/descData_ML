Understanding the Schizophrenia Prediction System

Project Overview

This web application uses machine learning to predict schizophrenia proneness based on clinical symptoms and demographic information. The system employs a Support Vector Machine (SVM) model trained on a comprehensive dataset of patient symptoms.

Dataset Information
  • Source: Schizophrenia Symptoms Dataset
  • Total Records: ~5000 patient records
  • Features: Age, Gender, Marital Status, Fatigue, Slowing, Pain, Hygiene, Movement
  • Target Classes: 5 levels of schizophrenia proneness
Model Details
  • Algorithm: Support Vector Machine (SVM) with RBF kernel
  • Preprocessing: StandardScaler normalization
  • Data Balancing: SMOTE (Synthetic Minority Oversampling Technique)
  • Validation: Train-test split (80-20)
